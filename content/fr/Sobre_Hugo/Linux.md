## Facilité
La première chose est d'installer Hugo et Visual Studio depuis notre terminal :

```shell
sudo apt-get install hugo
```
```shell
sudo snap install code --classic
```

## Le site Web
Ensuite, nous allons créer un site Web

```shell
hugo new site <nombre_del_sitio>
```
