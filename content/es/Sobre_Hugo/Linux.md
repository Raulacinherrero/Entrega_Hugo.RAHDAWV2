## Instalación
Lo primero es Instalar Hugo y Visual Studio desde nuestra terminal:

```shell
sudo apt-get install hugo
```
```shell
sudo snap install code --classic
```

## El Sitio Web
A continuación Crearemos un sitio Web

```shell
hugo new site <nombre_del_sitio>
```
